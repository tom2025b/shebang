# shebang

> **WARNING: May cause seizures, nostalgia, or terminal rage — run at own risk.**

An 80s disco terminal experience powered by Cyndi Lauper's *She Bop*.
Rainbow ANSI chaos. Lyrics everywhere. No apologies.

## What it does

- Clears your terminal and takes over
- Floods the screen with *She Bop* lyrics in cycling rainbow colors
- Scatters `she bop`, `you bop`, `everybody bops` like drunk pixels
- Loops forever until you hit **Ctrl+C**
- Fades out gracefully: *She bop… fades away.*

## Install

```bash
chmod +x shebang.sh

# Add the alias to your shell:
echo "alias shebang='bash /path/to/shebang.sh'" >> ~/.bashrc
source ~/.bashrc
```

## Usage

```bash
shebang
```

That's it. She bops.

## Requirements

- bash 4+
- A terminal that supports ANSI escape codes (basically all of them)
- A high tolerance for chaos
